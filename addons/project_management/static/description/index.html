Project Management Module

Comprehensive project management for Odoo 18 — Project Passport, Team management, Attachments, Tasks integration, and role-based access control.

Includes: Admin / Project Manager / Employee roles • Russian translations (ru.po) • Date validation • Docker-ready

Features

🎯 Project Passport

Project metadata: names (RU/EN), type, status, owner and manager, keywords, and links.

👥 Team & Roles

Manage members and roles (manager flag). Views honor permissions: admins, managers, employees.

📎 Attachments & Links

Store documents and project-related links (repo, docs, design, chat, meetings).

✅ Tasks Integration

Quick access to project tasks via the "Go to Tasks" action and header button.

🔒 Role-based Access

Security rules restrict editing to project managers; admins have full access.

🌐 Translations

Russian translations included in i18n/ru.po (labels, buttons, help texts).

⚙️ Installation

Copy the module into your addons path and update Apps; compatible with Odoo 18 (Docker compose recommended).
